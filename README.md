# Quote Generator

A simple web page to display random inspirational quotes.

## Getting Started

Just open the index.html file in your web browser.

## How It Works

The page will load a quote, and you can likely click a button to generate a new one.

## Technologies Used

* HTML
* CSS
* JavaScript
